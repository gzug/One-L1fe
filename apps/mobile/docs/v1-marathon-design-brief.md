# One L1fe — V1 Marathon Design Brief

Use the attached premium activity-app reference as the primary visual style, but do not copy the reference brand, logo, exact text, or layout.

The app must support both **Dark Mode** and **Light Mode**.

## Dark Mode

Dark Mode is the primary direction:

- graphite / near-black background
- soft glass-like dark cards
- subtle borders
- off-white typography
- coral primary accent
- clearly distinct golden amber secondary / synthetic accent
- circular or ring-like data presentation where useful
- premium, calm, athletic, classic

## Light Mode

Light Mode must also be polished:

- warm off-white / sand background
- clean white or warm-grey cards
- graphite text
- same coral primary accent
- same distinct golden amber secondary / synthetic accent
- same layout language as dark mode
- not colourful wellness, not clinical

## Colour rule

Increase contrast between the coral primary accent and the yellow/orange secondary accent.

The secondary colour should be more golden/amber and visibly different from coral.

## Keep

- Real Data / Demo Filled separation
- visible Light / Dark toggle
- synthetic/demo data amber-labelled in both modes
- live/real data neutral or coral, not amber
- no new icon libraries or design dependencies

## Do not

- copy the reference brand
- add unnecessary animation
- hide synthetic labels
- redesign app architecture
