# packages/domain

Shared domain layer for One L1fe.

This package should become the source of truth for:
- biomarker identifiers,
- units,
- normalization rules,
- validation schemas,
- typed contracts,
- recommendation and evidence structures.

## Why this exists

Without a domain layer, product rules get duplicated between the app, backend, and prompts. That becomes brittle very quickly.
