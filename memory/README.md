# memory/

Short-term working memory for One L1fe.

## Purpose

Use this folder for compact daily notes that help session continuity without polluting:
- `CHECKPOINT.md`, which is current execution truth,
- `MEMORY.md`, which is durable truth.

## What belongs here

- daily findings,
- unresolved questions,
- short-lived decisions,
- handoff notes,
- closeout snapshots that may matter for the next few days.

## What does not belong here

- raw personal health data,
- lab reports,
- screenshots with sensitive information,
- durable policy that belongs in `MEMORY.md`,
- current execution truth that belongs in `CHECKPOINT.md`.

## Naming

Use:
- `YYYY-MM-DD.md`

Example:
- `2026-04-13.md`

## Promotion rule

Promote notes out of this folder when they become:
- current truth, into `CHECKPOINT.md`, or
- durable truth, into `MEMORY.md`.
