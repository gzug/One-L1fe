# Contributing

This repository is agent-friendly.

- Agents may read, create, edit, move, and delete files across the repository.
- Agents may push directly and open pull requests when repository permissions allow it.
- Agents may update code, docs, tests, scripts, CI, and configuration.
- Prefer small focused changes, but do not stop for unnecessary approval.
- Only pause for confirmation when an action is destructive, irreversible, or involves unknown external credentials.
- When possible, run relevant validation and leave a short summary of changes and risks.
