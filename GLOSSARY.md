# GLOSSARY.md

Abbreviations and terms used in One L1fe (OL). Update when new recurring abbreviations enter the repo.

| Abbreviation | Full Term | Category | Notes |
| --- | --- | --- | --- |
| OC | OpenClaw | Agent Platform | Local agent runtime and orchestration layer |
| OL | One L1fe | Product | Personal health-tracking product and primary repo |
| PC | Paperclip AI | External Tool | Referenced AI tool or team context. Confirm operational role before integrating into workflows |
| GH | GitHub | Platform | Source control, collaboration, remote hosting |
| DA | Digital Avatar | Product Concept | Long-term personalized representation built from health data, habits, and biomarker patterns |
| BM | Biomarker | Health Data | Measurable lab or physiological marker used for tracking, correlation, and trend analysis |
| DSGVO | Datenschutz-Grundverordnung (GDPR) | Compliance | EU data-protection framework; relevant for special-category health data under Art. 9 |
| MCP | Model Context Protocol | Architecture | Protocol for connecting models to tools, data sources, and structured context |
| RAG | Retrieval-Augmented Generation | Architecture | Pattern for grounding model outputs in stored documents, records, or evidence |
| MVP | Minimum Viable Product | Delivery | Narrowest credible first version testable with real users and real workflows |
| API | Application Programming Interface | Engineering | Interface used to connect app, backend, and AI services |
| ApoB | Apolipoprotein B | Biomarker | Core lipid-related biomarker in the MVP set |
| LDL | Low-Density Lipoprotein | Biomarker | Core lipid biomarker; tracked with ApoB and triglycerides |
| Lp(a) | Lipoprotein(a) | Biomarker | Core inherited cardiovascular risk biomarker in the MVP set |
| HbA1c | Hemoglobin A1c | Biomarker | Core long-range glucose marker for metabolic tracking |
| CRP | C-reactive Protein | Biomarker | Core inflammation-related biomarker in the MVP set |
| DAO | Diamine Oxidase | Biomarker | Contextual biomarker; treated as optional and interpretation-sensitive |
| B12 | Vitamin B12 | Biomarker | Supporting biomarker for nutrient status context |
